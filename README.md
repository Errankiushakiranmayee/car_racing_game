# Car Racing Game using Scratch

## Description
This is a simple car racing game developed using Scratch.
The player controls a car using keyboard keys and avoids obstacles.
The score increases as the player survives longer.

## Controls
- Left Arrow: Move Left
- Right Arrow: Move Right

## Files
- car_racing_game.sb3 – Scratch project file
- Screenshots – Game preview images

## How to Run
1. Go to https://scratch.mit.edu
2. Click File → Load from your computer
3. Open the car_racing_game.sb3 file
